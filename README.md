# EventBoard: RTC-Driven Message Display System

## Overview

EventBoard is a real-time embedded system developed using LPC2148 ARM7 Microcontroller.

The system automatically displays predefined messages on a 16x2 LCD based on RTC time and provides secure admin access for time editing and message selection.

## Features

- RTC Based Message Scheduling
- Password Protected Admin Mode
- LCD Scrolling Messages
- LM35 Temperature Monitoring
- LED Status Indication
- Keypad Based User Interface

## Hardware

- LPC2148
- RTC
- LCD 16x2
- 4x4 Matrix Keypad
- LM35
- LEDs
- Buzzer

## Software

- Embedded C
- Keil uVision
- Flash Magic
